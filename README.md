### Building the mod yourself
- Clone the repository
- Run `./gradlew build`

### Installation
- Install [Fabric](https://fabricmc.net/use/installer/) as the Modloader
- Put the mod in your mods folder
- Put [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api) in you mods folder
- You are good to go!

### Features
- Flight (Lets you in every Gamemode. Also works with Entitys that can be Ridden)
- Nofall/Antifall (Makes you take no falldamage)
- Velocity/Antiknockback (Makes you take no knockback)
- Clipper/Teleporter (Teleports you in the given direction)
- Speed (Makes you faster)
- SpeedHop (Makes you faster and makes you jump)
- FreeCam (Allows you to Fly through Blocks | Inspired by hashalite's Freecam -> https://github.com/hashalite/Freecam)
- Vision (Makes Entitys visible through walls)
- Xray (See Ores through Blocks)
- Reach (Makes you able to hit entities from further away)
- KillAura (Automaticly attacks entities for you)
- MusicPlayer (Play the ingame Discs Clientside without placing a Jukebox | use "^playsong list" to list all available soundtracks)

### Features

- Coordinates Hud [Shows your Coordinates]

- Direction Hud [Shows your direction]

- No Break Particles [Removes the Clientside break Particles]

- No Explosion Particles [Removes the Clientside Explosion Particles]

- Built in Texture Pack for Gui ELements [Can be overridden]
